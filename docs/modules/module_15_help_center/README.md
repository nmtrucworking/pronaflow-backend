# Module 15: Help Center & Knowledge Base

## Overview
Self-service knowledge base and help system:
- Article management with versioning
- Multi-language support
- SEO optimization
- Search functionality
- Article feedback
- Help center analytics

## Key Components

### Models
- **Article** - Knowledge base articles
- **ArticleVersion** - Version history
- **ArticleTranslation** - Multi-language support
- **Category** - Article categorization
- **ArticleFeedback** - User feedback on articles
- **FailedSearch** - Improve search relevance

### Services
- `ArticleService` - Article CRUD and publishing
- `SearchService` - Full-text search
- `TranslationService` - Multi-language management
- `AnalyticsService` - Help center metrics

## Database Tables (6)
- articles
- article_versions
- article_translations
- categories
- article_feedbacks
- failed_searches

## Features
- Rich text editor with markdown support
- Version history and rollback
- Multi-language translations
- Full-text search
- Article ratings and feedback
- SEO-friendly URLs
- Analytics on popular articles
- Search analytics for improvement
